# AutoFIR

An AI-powered FIR (First Information Report) auto-draft generator. 

## Features
- Plain-English crime complaint to formally structured FIR.
- Automated IPC and BNS 2023 section mapping.
- Print-ready PDF export.

## Setup
1. `pip install -r requirements.txt`
2. Copy `.env.example` to `.env` and fill in your Groq API Key and MongoDB URI.
3. Run `streamlit run main.py`
